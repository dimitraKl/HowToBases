Τεχνική Αναφορά

Η εφαρμογή μας καλύπτει την απαιτούμενη λειτουργικότητα της
ομαδικής εργασίας. Συγκεκριμένα, στην βάση δεδομένων μας ο
χρήστης μπορεί να δημιουργήσει έναν πίνακα και να ορίσει τα
πεδία του, ενώ ως επιπρόσθετη λειτουργία παρέχεται ο ορισμός
ενός πεδίου ως σημείο αναφοράς, βάσει του οποίου γίνεται η
αναζήτηση στον συγκεκριμένο πίνακα. Επίσης, ο χρήστης μπορεί
να εισάγει, να διαγράψει, να τροποποιήσει, καθώς και να
εμφανίσει τα δεδομένα που εκείνος επιθυμεί.Επιπροσθέτως, ο
χρήστης έχει τη δυνατότητα να εισάγει πολλαπλούς πίνακες
στην βάση δεδομένων του τους οποίους επίσης μπορεί να
διαγράψει, να εμφανίσει ή να τροποποιήσει, ενώ η βάση που
δημιουργεί αποθηκεύεται σε ένα αρχείο CSV. Με τον τρόπο αυτό
μπορεί να επεξεργαστεί περαιτέρω την βάση του χρησιμοποιώντας
το πακέτο excel επί του αρχείου αλλά και να ξανατρέξει το
πρόγραμμα φορτώνοντας την ήδη υπάρχουσα βάση του για 
επεξεργασία με τις λειτουργίες που παρέχει η εφαρμογή μας.
Όλα αυτά πραγματοποιούνται σε ένα Graphic User Interface (GUI)
που έχουμε δημιουργήσει για να είναι φιλικότερη η εφαρμογή
προς τον χρήστη.
